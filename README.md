oxmem-fuse
===============

A wrapper for ETRI OmniXtend Memory device.

libfuse-dev package is needed.

\> sudo apt install libfuse-dev

Usage
-----

\> make

\> make test

\> ./oxmem-mount [mount point]

\> fusermount [mount point]
